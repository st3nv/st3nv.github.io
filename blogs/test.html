# Easiest way to make a webpage from markdown Author: TS, Date: 20th July 2024
I have been exploring ways to make a webpage from markdown for a while now. From simply copying the rendered html from markdown editor to using Jekyll. But they either need a lot of pre-configurations or a pain to write and maintain. Until recently I came across this Github Repo [md-page](https://github.com/oscarmorrison/md-page) by Oscar Morrison. And I am shocked by the simplicity, convenience and the flexibility it offers.

For those who want to start blogging or share a webpage without much hassle, here is a step by step guide of it. As you may notice, this page is also made using md-page, with a little bit of customized css based on the [Splendor](https://github.com/markdowncss/splendor) theme by John Otander.
