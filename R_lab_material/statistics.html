7. Statistical analysis in R

R is especially handy when it comes to statistical analysis of data

7.1. t test

As you have learnt in class, here we show how to perform t-test in R.

data(iris)
t.test(iris$Sepal.Length, iris$Sepal.Width)

7.2 Simple linear regression

Doing linear regression in R is very easy. We just need to use lm and specify the dependent and independent variables.

Here is an example of simple linear regression, where we are trying to predict Sepal Length using Sepal Width. We saw the operator ~ before in creating plots. It's called formula method.

data(iris)
lm1 <- lm(iris$Sepal.Length ~ iris$Sepal.Width)

Then we can use summary to get the summary of the model.

summary(lm1)

And we can use predict to predict the values of Sepal Length using Sepal Width. For example, if we want to predict the Sepal Length when Sepal Width is 3.5, we can do the following.

predict(lm1, 3.5)

7.3 Multiple linear regression

To perform multiple linear regression in R, we need to include all the variables in the model.

data(iris)
lm(iris$Sepal.Length ~ iris$Sepal.Width + iris$Petal.Length + iris$Petal.Width)

7.4 ANOVA

As you have learnt in class, here we show how to perform ANOVA in R.

data(iris)
aov(iris$Sepal.Length ~ iris$Species)
