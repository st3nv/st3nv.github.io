2. Reading Data

There are many ways to read data in R. Here we show how to read data from a CSV file, an Excel file, and a Dat file.

You will need to use the path to the file on your computer, or on the Internet.

If you don't know how to get the path to the file...

  1. For windows users, just click the top bar of your file explorer and copy it.
  2. For Mac users, right click on the file and select "Get Info". Right click "Where" and copy as pathname. Or you can press Option+Command+P to show the path bar, then right click and select "Copy ... as pathname"

2.1 CSV files

URL = "https://gist.githubusercontent.com/netj/8836201/raw/6f9306ad21398ea43cba4f7d537619d0e07d5ae3/iris.csv"
x <- read.csv(URL)

# or

x <- read.table(URL, sep = ",", header = TRUE)
head(x)

2.2 Excel files

install.packages("readxl")
library(readxl)
xlsx_example <- readxl_example("datasets.xlsx")
x <- read_excel(xlsx_example)
head(x)

If you want...

You can use file.choose() instead of path to file to select your file with a GUI. But you have to do it everytime you run the code.

x <- read.csv(file.choose())
