1.2. Variables

To assign a value to a variable, use the <- operator. The = operator can also be used, but it is not recommended. 

x <- 5
y = 6
x + y

text <- "Hello World!"

# paste() and paste0() function can be used to concatenate strings and variables, 
# run the following and see what happens

print(paste(text, x))
print(paste0(text, x))

1.3. Vectors

Vectors are the most basic data type in R. They can be numeric, character, or logical. 

x <- c(1, 2, 3, 4, 5)
y <- c(6, 7, 8, 9, 10)

# the operations are conducted element-wise
x + y

# It can also be a character vector
x <- c('1', '2', '3', '4', '5')

# or logical
x <- c(TRUE, FALSE, TRUE, FALSE, TRUE)

1.4. Matrices

Matrix is also a basic data type in R. It is a two-dimensional array of data. The elements in a matrix must be of the same data type.

In this course probably won't use matrices much, but it is good to know how to create them. 

x <- matrix(c(1, 2, 3, 4, 5, 6), nrow = 2, ncol = 3)
y <- matrix(c(7, 8, 9, 10, 11, 12), nrow = 3, ncol = 2)

# Perform matrix multiplication
x %*% y

1.5. Data Frames

Data frames are the most commonly used data type in R. They are similar to tables in Excel. They are two-dimensional arrays of data. The elements in a data frame can be of different data types. 

# Define a data frame manually
x <- data.frame(name = c("John", "Mary", "Peter"), age = c(20, 21, 22))
x

# Accessing elements
x[1, 1]
x[1:2, ]
x[c(1, 3), c(1, 2)]
x$age

There are many ways to create a data frame. One of the most common ways is to read a data file into R. We will learn how to do that in the next section.

1.6. factors

Factors are used to represent categorical variables with limited number of unique values. Like " Male" or "Female" 

x <- factor(c("Male", "Female", "Male", "Female", "Male"))

There are also many other datatype in R, such as lists, arrays, and data tables. You can explore them on your own.
